# Dokumentasi Strategi Berkah Utami Furniture / bufinterior.com

Folder ini berisi dokumentasi arah besar website `bufinterior.com`.

Tujuan dokumentasi ini:
1. Menjaga arah website tetap konsisten.
2. Membantu assistant/developer berikutnya memahami konteks tanpa mengulang riset dari nol.
3. Menjadi pegangan sebelum perubahan SEO, konten, struktur URL, halaman jasa, portofolio, dan Google Business Profile.
4. Mencegah keputusan berbahaya seperti doorway page, keyword stuffing, halaman area massal, atau perubahan URL sembarangan.

## Dokumen Utama

- `blueprint/README.md`  
  Index utama blueprint.

- `blueprint/01-strategy-foundation.md`  
  Visi, positioning, target market, 3 pilar layanan, dan prinsip keputusan.

- `blueprint/02-seo-architecture-crawler.md`  
  Struktur URL, aturan `/layanan/*` dan `/jasa/*`, prinsip 3 klik crawler, sitemap, schema, dan teknis SEO.

- `blueprint/03-keyword-serp-priorities.md`  
  Ringkasan hasil riset manual Google lokal dan prioritas halaman P0/P1.

- `blueprint/04-content-photo-whatsapp-sop.md`  
  SOP konten, foto portofolio, WhatsApp conversion, dan CTA.

- `blueprint/05-roadmap-risk-register.md`  
  Roadmap 30/60/90 hari, risiko, mitigasi, dan keputusan go-live.

- `blueprint/06-assistant-handoff.md`  
  Ringkasan wajib baca untuk assistant/developer yang melanjutkan proyek.

## Prinsip Utama

Website ini bukan sekadar company profile. Website ini adalah aset growth dan lead generation lokal untuk Berkah Utami Furniture.

Fokus utama:
- Google Search
- Google Maps / Google Business Profile
- Google Images
- Portofolio portfolio
- Artikel SEO pendukung
- Konversi WhatsApp

Target utama bukan hanya traffic, tetapi calon pelanggan yang punya kebutuhan proyek nyata.
