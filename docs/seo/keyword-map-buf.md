# Keyword Map BUF — SEO Direction

Dokumen ini menjadi acuan keyword website BUF agar strategi SEO tidak terlalu sempit ke satu lokasi saja.

## Prinsip Utama

BUF tidak diposisikan sebagai bisnis yang hanya melayani satu area. BUF diposisikan sebagai workshop interior dan custom furniture yang membantu kebutuhan ruang sesuai ukuran, fungsi, material, dan budget.

Lokasi tetap dipakai untuk SEO lokal, tetapi tidak mendominasi semua headline dan copy utama.

## Strategi Keyword

| Kelompok | Fungsi | Contoh keyword | Penempatan |
|---|---|---|---|
| Service-first | Menjelaskan layanan utama | kitchen set custom, custom furniture, backdrop TV custom | H1, hero, halaman layanan |
| Problem-first | Menjawab masalah user | biaya kitchen set custom, material kitchen set yang bagus | Artikel, FAQ |
| Buyer-intent | Menangkap user siap kontak | buat kitchen set custom, pesan lemari custom, jasa buat booth usaha | CTA, halaman layanan, kontak |
| Local-support | Mendukung local SEO | kitchen set Banyumas, custom furniture Purwokerto | Metadata, slug, schema, GBP, sebagian kecil body copy |

## Aturan Copy

| Elemen | Aturan |
|---|---|
| H1 / Hero | Utamakan manfaat dan layanan, jangan terlalu lokal |
| Meta title | Boleh mengandung keyword lokal |
| Meta description | Campur layanan + manfaat + lokasi by konfirmasi |
| Body copy | Service-first, jangan mengulang lokasi berlebihan |
| Footer | Brand + layanan utama, lokasi cukup sebagai konteks kecil |
| FAQ | Jawab pertanyaan user, bukan hanya menjejalkan keyword |
| Artikel | Fokus problem dan edukasi |
| Gallery | Fokus visual dan kategori layanan |

## Keyword yang Jangan Dikejar Dulu

| Keyword | Alasan |
|---|---|
| jasa interior Jakarta | Tidak relevan dengan kapasitas awal |
| desain interior mewah | Terlalu premium/luxury, belum sesuai positioning |
| interior apartemen | Belum jadi layanan utama |
| kontraktor interior nasional | Terlalu luas |
| area page banyak kecamatan | Risiko doorway page jika tanpa proyek unik |

## Rekomendasi Metadata

### Homepage

- Primary: interior custom
- Secondary: custom furniture, kitchen set custom, jasa interior custom
- Local support: berbasis workshop, lokasi proyek by konfirmasi

### Halaman Layanan

Gunakan pola:

`[Layanan Utama] Custom | Desain & Pengerjaan Sesuai Ukuran — BUF`

Contoh:

`Kitchen Set Custom | Desain & Pengerjaan Sesuai Ukuran — BUF`

Local keyword boleh muncul di variasi meta jika dibutuhkan, tetapi jangan semua title harus terasa lokal.

### Artikel

Gunakan pola:

`[Masalah/Pertanyaan User] | Panduan BUF`

Contoh:

`Biaya Kitchen Set Custom: Faktor yang Mempengaruhi Harga | BUF`

## Checklist Sebelum Online

- [ ] Pastikan metadata tidak terlalu location-heavy.
- [ ] Pastikan sitemap memuat artikel.
- [ ] Pastikan semua layanan punya CTA WhatsApp.
- [ ] Pastikan gallery punya kategori yang mudah diupdate.
- [ ] Pastikan Google Business Profile disiapkan.
- [ ] Pastikan Search Console submit sitemap setelah domain final.
- [ ] Pastikan domain final menggantikan `pages.dev` di `content/site.js`.
- [ ] Pastikan artikel pertama diperkaya sebelum indexing serius.
- [ ] Pastikan foto proyek asli ditambahkan bertahap.

## Catatan Final

Keyword lokal tetap penting, tetapi tidak boleh membuat BUF terasa hanya melayani satu lokasi. Website harus terasa sebagai brand interior/custom furniture yang profesional, terbuka untuk konsultasi proyek, dan tetap punya fondasi local SEO yang kuat.
