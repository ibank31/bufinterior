# Keyword Map BUF — SEO Direction

Dokumen ini menjadi acuan keyword website BUF agar strategi SEO tidak terlalu sempit ke satu lokasi saja.

## Prinsip Utama

BUF tidak diposisikan sebagai bisnis yang hanya melayani satu area. BUF diposisikan sebagai workshop interior dan custom furniture yang membantu kebutuhan ruang sesuai ukuran, fungsi, material, dan budget.

Lokasi tetap dipakai untuk SEO lokal, tetapi tidak mendominasi semua headline dan copy utama.

Strategi keyword harus natural. Keyword dipakai untuk menjawab pertanyaan user, bukan diulang secara spam.

## Strategi Keyword

| Kelompok | Fungsi | Contoh keyword | Penempatan |
|---|---|---|---|
| Service-first | Menjelaskan layanan utama | kitchen set custom, custom furniture, backdrop TV custom | H1, hero, halaman layanan |
| Problem-first | Menjawab masalah user | biaya kitchen set custom, material kitchen set yang bagus | Artikel, FAQ, estimasi |
| Buyer-intent | Menangkap user siap kontak | buat kitchen set custom, pesan lemari custom, jasa buat booth usaha | CTA, halaman layanan, kontak |
| Price-intent | Menangkap user yang cari biaya | estimasi biaya interior custom, harga lemari custom, biaya kitchen set custom | `/estimasi-biaya`, artikel biaya |
| Trust-intent | Menangkap user yang membandingkan vendor | tips memilih jasa interior custom, vendor interior, workshop interior | Artikel, tentang kami, gallery |
| Local-support | Mendukung local SEO | kitchen set Banyumas, custom furniture Purwokerto | Metadata, slug, schema, GBP, sebagian kecil body copy |

## Aturan Copy

| Elemen | Aturan |
|---|---|
| H1 / Hero | Utamakan manfaat dan layanan, jangan terlalu lokal |
| Meta title | Service-first, local keyword boleh dipakai terbatas |
| Meta description | Campur layanan + manfaat + lokasi by konfirmasi |
| Body copy | Service-first, jangan mengulang lokasi berlebihan |
| Footer | Brand + layanan utama + estimasi + artikel prioritas |
| FAQ | Jawab pertanyaan user, bukan menjejalkan keyword |
| Artikel | Fokus problem, edukasi, dan keputusan user |
| Gallery | Fokus visual dan kategori layanan |
| Estimasi | Transparan, range awal, bukan harga final |
| Google Business Profile | Tempat utama untuk local SEO |

## Checklist Sebelum Online

- [x] Metadata tidak terlalu location-heavy.
- [x] Sitemap memuat artikel.
- [x] Sitemap memuat `/estimasi-biaya`.
- [x] Semua layanan punya CTA WhatsApp.
- [x] Gallery punya struktur kategori yang mudah diupdate.
- [x] Artikel awal sudah diperkaya.
- [x] Internal linking cluster sudah diperkuat.
- [ ] Google Business Profile disiapkan.
- [ ] Search Console submit sitemap setelah domain final.
- [ ] Domain final menggantikan `pages.dev` di `content/site.js`.
- [ ] Foto proyek asli ditambahkan bertahap.
- [ ] Audit gambar sebelum launch final.
- [ ] Tambah trust proof/testimoni section.

## Catatan Final

Keyword lokal tetap penting, tetapi tidak boleh membuat BUF terasa hanya melayani satu lokasi. Website harus terasa sebagai brand interior/custom furniture yang profesional, terbuka untuk konsultasi proyek, punya halaman estimasi yang jujur, dan tetap punya fondasi local SEO yang kuat.
