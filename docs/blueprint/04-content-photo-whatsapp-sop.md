# 04 - Content, Foto Portfolio, Gallery, dan WhatsApp SOP

## 1. Prinsip Konten

Konten BUF harus:
- menjawab pertanyaan calon pelanggan
- menunjukkan bukti pekerjaan
- mengarahkan ke WhatsApp
- memperkuat halaman jasa
- tidak generik
- tidak terlalu menjual tanpa bukti
- tidak mengarang harga
- tidak memakai klaim berlebihan

## 2. Fungsi Konten per Jenis Halaman

### Homepage

Fungsi:
- menjelaskan siapa BUF
- menampilkan 3 pilar layanan
- menonjolkan workshop Berkoh
- memberi link cepat ke 5 halaman P0
- memberi CTA WhatsApp
- menampilkan gallery ringkas
- mengarahkan ke Google trust signal

### /layanan/*

Fungsi:
- edukasi layanan
- material
- proses
- FAQ
- internal link ke /jasa/*

### /jasa/*

Fungsi:
- menangkap keyword lokal
- menjawab intent beli
- menampilkan trust signal lokal
- menunjukkan bukti pekerjaan
- mengarahkan ke WhatsApp

### /gallery

Fungsi:
- bukti visual
- sumber internal link ke halaman uang
- pendukung Google Images
- pendukung trust calon pelanggan

### /artikel

Fungsi:
- menjawab long-tail query
- mendukung halaman layanan dan jasa
- mengalirkan traffic ke CTA

## 3. Framework Landing Page /jasa/*

Setiap halaman /jasa/* wajib memiliki:
1. Hero dengan H1 jelas.
2. Copy singkat yang menyebut layanan, Purwokerto/Banyumas, dan workshop Berkoh.
3. CTA WhatsApp spesifik.
4. Masalah umum pelanggan.
5. Solusi BUF.
6. Jenis pengerjaan.
7. Opsi material atau pengerjaan.
8. Gallery atau proof.
9. Proses pemesanan.
10. Area layanan.
11. FAQ.
12. CTA akhir.
13. Internal link ke /layanan/*, /gallery, /estimasi-biaya, /kontak.

## 4. SOP Foto Proyek

Setiap proyek harus didokumentasikan.

Minimal foto:
- sebelum pengerjaan
- proses workshop
- proses pemasangan
- detail material
- detail finishing
- hasil akhir dari jauh
- hasil akhir dari dekat
- foto vertikal untuk sosial media
- foto horizontal untuk website

Prioritas foto:
1. foto realisasi asli
2. foto proses pengerjaan
3. foto detail material/finishing
4. foto workshop
5. konsep atau render hanya jika diberi label jelas sebagai konsep

## 5. Standar Nama File

Format:
    layanan-area-jenis-buf-urutan.webp

Contoh:
    kitchen-set-custom-purwokerto-realisasi-buf-01.webp
    booth-usaha-custom-purwokerto-realisasi-buf-01.webp
    akrilik-custom-display-purwokerto-proses-buf-01.webp
    etalase-aluminium-purwokerto-realisasi-buf-01.webp
    kanopi-rumah-purwokerto-pemasangan-buf-01.webp

Jangan pakai:
    IMG_1234.jpg
    WhatsApp Image 2026-06-06.jpg
    foto1.jpg
    final_fix_baru.jpg

## 6. Standar Alt Text

Alt text harus:
- deskriptif
- natural
- tidak keyword stuffing
- menyebut layanan
- menyebut area jika relevan
- menjelaskan isi foto

Contoh:
- Kitchen set custom modern untuk dapur rumah di Purwokerto oleh Berkah Utami Furniture.
- Booth usaha custom untuk UMKM di Purwokerto dengan display produk dan area branding.
- Etalase aluminium custom untuk toko di Purwokerto.
- Proses pemasangan kanopi rumah di area Purwokerto.

## 7. Standar Caption

Caption harus menjelaskan:
- jenis pekerjaan
- konteks proyek
- area umum
- material atau fitur jika aman
- status foto: realisasi, proses, atau konsep

Contoh:
Realisasi kitchen set custom untuk dapur rumah di area Purwokerto. Ukuran, finishing, dan penyimpanan menyesuaikan kebutuhan pelanggan.

Contoh:
Proses pengerjaan booth usaha custom dari workshop Berkah Utami Furniture untuk kebutuhan display produk UMKM.

## 8. Label Foto

Gunakan label:
- Realisasi
- Proses
- Konsep

Aturan:
- Jangan label foto konsep sebagai realisasi.
- Jangan pakai gambar AI sebagai bukti proyek.
- Foto render boleh, tapi harus jelas sebagai konsep.
- Foto stok internet tidak boleh dipakai sebagai portfolio.

## 9. Watermark

Watermark boleh digunakan:
- kecil
- rapi
- di pojok
- tidak mengganggu visual
- tidak menutupi detail pekerjaan

Yang lebih penting daripada watermark:
- nama file
- alt text
- caption
- halaman tujuan
- konsistensi visual

## 11. WhatsApp CTA

Jangan pakai CTA generik seperti “Hubungi Kami” saja.

Gunakan:
- Konsultasi Kitchen Set via WhatsApp
- Konsultasi Booth Usaha via WhatsApp
- Konsultasi Akrilik Custom via WhatsApp
- Konsultasi Aluminium via WhatsApp
- Konsultasi Kanopi via WhatsApp
- Konsultasi Ukuran & Estimasi via WhatsApp

## 12. Prefilled WhatsApp Message

Kitchen Set:
Halo Berkah Utami Furniture, saya ingin konsultasi pembuatan kitchen set custom di area Purwokerto/Banyumas. Saya ingin tanya estimasi dan opsi materialnya.

Booth Usaha:
Halo Berkah Utami Furniture, saya ingin konsultasi pembuatan booth usaha custom di Purwokerto. Rencananya untuk usaha saya. Bisa dibantu estimasi?

Akrilik Custom:
Halo Berkah Utami Furniture, saya ingin konsultasi akrilik custom untuk kebutuhan display/papan nama/rak usaha. Bisa dibantu estimasi?

Aluminium:
Halo Berkah Utami Furniture, saya ingin konsultasi pekerjaan aluminium custom seperti kusen/pintu/jendela/etalase/partisi di area Purwokerto. Bisa dibantu?

Kanopi:
Halo Berkah Utami Furniture, saya ingin konsultasi pembuatan kanopi untuk rumah/toko di area Purwokerto/Banyumas. Bisa dibantu estimasi?

## 13. Pertanyaan Admin Pertama

Admin harus menanyakan:
1. Lokasi proyek di mana?
2. Layanan yang dibutuhkan apa?
3. Ukuran perkiraan berapa?
4. Ada foto lokasi?
5. Ada referensi model?
6. Kapan rencana pengerjaan?
7. Budget ingin diarahkan ekonomis, standar, atau premium?

## 14. Cara Tanya Budget dengan Halus

Gunakan:
Agar kami bisa arahkan material dan desain yang paling pas, kisaran budget ingin diarahkan ke ekonomis, standar, atau premium ya?

Jangan gunakan:
Budget berapa?

## 15. Follow Up

H+1:
Halo Kak, kami follow up untuk rencana pengerjaan [layanan]. Jika sudah ada foto lokasi atau ukuran kasar, bisa dikirim agar kami bantu arahkan estimasi awal.

H+3:
Halo Kak, apakah rencana [layanan] masih ingin dilanjutkan? Jika belum urgent, tidak apa-apa. Kami bisa bantu simpan catatan kebutuhannya dulu untuk estimasi berikutnya.

## 16. Data Lead yang Wajib Dicatat

Format:
- tanggal
- nama
- nomor WA
- layanan
- lokasi
- ukuran atau perkiraan kebutuhan
- budget range jika ada
- sumber lead
- status
- follow-up berikutnya
- catatan admin

## 17. Sumber Ide Artikel dari Chat WhatsApp

Pertanyaan calon pelanggan harus dicatat sebagai bahan artikel.

Contoh pertanyaan yang bisa jadi artikel:
- kitchen set aluminium atau plywood lebih bagus?
- berapa biaya kitchen set per meter?
- booth portable cocok untuk usaha apa?
- akrilik susu dan akrilik bening bedanya apa?
- kusen aluminium cocok untuk rumah lembap?
- kanopi alderon atau spandek lebih bagus?

Artikel dibuat untuk menjawab pertanyaan nyata, bukan sekadar mengejar keyword.
